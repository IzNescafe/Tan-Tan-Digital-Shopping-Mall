# Web App

This folder now contains the starter React frontend for Tan Tan.

## Theme foundation

The app uses a shared pastel brand system based on:

- `#CDB4DB`
- `#FFC8DD`
- `#FFAFCC`
- `#BDE0FE`
- `#A2D2FF`

The core theme is defined in `src/styles/theme.css`.

## Current setup

- React with Vite
- Global brand tokens and reusable surface styles
- Starter landing page sections for marketplace, dashboard, and workflow previews

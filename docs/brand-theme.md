# Brand Theme

Tan Tan now uses a soft pastel visual system based on the following palette:

- `#CDB4DB` thistle
- `#FFC8DD` fairy tale
- `#FFAFCC` carnation pink
- `#BDE0FE` uranian blue
- `#A2D2FF` light sky blue

## Theme direction

- Soft, welcoming, and premium rather than harsh or corporate
- Trust-first surfaces for shopping, proof verification, and order tracking
- Rounded cards and layered glassmorphism for marketplace and dashboard screens
- Warm editorial typography paired with a clean sans-serif UI font

## Usage rules

- Use `#CDB4DB` and `#FFC8DD` for warm brand atmosphere and soft highlights
- Use `#FFAFCC` for primary emphasis and call-to-action gradients
- Use `#BDE0FE` and `#A2D2FF` for secondary accents, dashboard states, and cool balance
- Keep core text dark and neutral for readability
- Prefer gradients, translucent surfaces, and gentle shadows over flat heavy blocks

## Implementation

The first implementation lives in `apps/web/src/styles/theme.css` and is intended to be the default styling foundation for:

- landing pages
- auth flows
- product catalog pages
- request submission screens
- retailer dashboards
- admin dashboards
